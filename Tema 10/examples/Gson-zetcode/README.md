# Java-Gson-Examples
Java Gson examples show how to work with JSON in Java using Gson library. 
We use three different Gson APIs to work with JSON. Gson is a Java library which converts Java Objects 
into their JSON representation and vice versa.

Examples com from the Java Gson tutorial
http://zetcode.com/java/gson/

1. GsonToJson
2. GsonFromJson
3. GsonBuilderEx
4. GsonPrettyPrinting
5. GsonSerializeNulls
6. GsonWriteList / GsonReadList
7. GsonReadArray
8. GsonReadWebPage
9. GsonExcludeFields
10. GsonDataBindApiWrite
11. GsonDataBindApiRead
12. GsonTreeModelWrite
13. GsonTreeModelRead
14. GsonStreamApiWrite
15. GsonStreamApiRead
